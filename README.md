# Rise & Thrive – Fitness Web Application

## 📌 Description
A web-based fitness application that promotes healthy lifestyle habits through challenges and reward-based systems.

## 🚀 Features
- User registration and login system
- Reward-based engagement system
- Fitness tracking and challenges
- Responsive user interface

## 🛠 Technologies Used
- PHP
- MySQL
- HTML, CSS
- JavaScript

## 🔐 Authentication
Implemented secure session-based login system.
